# Parser

Python-парсер для сбора русскоязычных новостей из веб-источников и сохранения результата в файл `news.xml`.

## Возможности

- выбор источника при запуске: `lenta`, `rbc`, `ria`;
- выбор количества новостей;
- сохранение полей `id`, `title`, `url`, `date`, `category`, `text`;
- вывод прогресса в консоль;
- промежуточное сохранение результата;
- задержка между запросами, чтобы не отправлять запросы слишком часто.

## Запуск

```bash
uv run python main.py
```

После запуска программа спросит:

```text
Введите источник:
Сколько новостей собрать:
```

Пример:

```text
ria
100
```

## Файлы проекта

- `main.py` — основной код парсера;
- `news.xml` — итоговый XML-файл с новостями;
- `pyproject.toml` — зависимости проекта;
- `uv.lock` — зафиксированные версии зависимостей;
- `.python-version` — версия Python для проекта;
- `.venv` — виртуальное окружение;
- `__pycache__` — автоматически созданный Python-кэш.
