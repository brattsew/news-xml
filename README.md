# News Parser

Python-парсер для сбора русскоязычных новостей из веб-источников и сохранения датасета в формате XML.

Проект сделан для практической работы по сбору данных из веб-источников. Код находится в одном файле `main.py`. 

## Что делает программа

Парсер:

- собирает ссылки на новости;
- открывает каждую новость;
- достаёт заголовок, ссылку, дату, категорию и полный текст;
- сохраняет результат в файл `news.xml`;
- выводит прогресс в консоль;
- делает задержку между запросами;
- периодически сохраняет промежуточный результат.

## Поддерживаемые источники

При запуске можно выбрать один из источников:

```text
lenta
rbc
ria
```

Источники:

- `lenta` — новости с сайта `lenta.ru`;
- `rbc` — новости с сайта `rbc.ru`;
- `ria` — новости с сайта `ria.ru`.

## Структура XML

Итоговый файл называется `news.xml`.

Пример структуры:

```xml
<dataset>
    <news id="1">
        <title>Заголовок новости</title>
        <url>https://example.com/news</url>
        <date>2026-04-26</date>
        <category>Политика</category>
        <text>Полный текст новости...</text>
    </news>
</dataset>
```

## Установка через uv

Если установлен `uv`, достаточно выполнить:

```bash
uv sync
```

Запуск:

```bash
uv run python main.py
```

## Установка через pip

Можно запустить проект и без `uv`.

Создать виртуальное окружение:

```bash
python -m venv .venv
```

Активировать окружение на macOS / Linux:

```bash
source .venv/bin/activate
```

Активировать окружение на Windows:

```bash
.venv\Scripts\activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить:

```bash
python main.py
```

## Как пользоваться

После запуска программа спросит источник:

```text
Источники: lenta, rbc, ria
Введите источник:
```

Например:

```text
ria
```

Потом программа спросит количество новостей:

```text
Сколько новостей собрать:
```

Например:

```text
100
```

После этого начнётся сбор данных. В консоли будет отображаться прогресс:

```text
Ссылок собрано: 100
Ссылок проверено: 10 | новостей сохранено: 10
```

## Настройки в коде

В начале `main.py` есть несколько настроек:

```python
OUTPUT_FILE = "news.xml"
SAVE_EVERY = 100
DELAY = 0.05
```

Что они означают:

- `OUTPUT_FILE` — имя итогового XML-файла;
- `SAVE_EVERY` — как часто сохранять промежуточный результат;
- `DELAY` — задержка между запросами.
